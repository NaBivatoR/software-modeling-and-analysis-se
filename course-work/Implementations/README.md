## Кратко описание
Проект: **HBO MAX** — имитирана платформа за стрийминг, включваща потребители, профили, планове за абонаменти, съдържание (филми/сериали), епизоди, жанрове, устройства, история на гледанията, плащания и ревюта. Проектът включва концептуален модел, логически модел, Data Warehouse схема и SQL имплементация.

## Инсталация и стартиране
Импортирайте SQL файловете в **MySQL Workbench** в следния ред:  
1. `create_tables.sql`  
2. `insert_data.sql`  
3. `procedure.sql`  
4. `function.sql`  
5. `trigger.sql`  

След изпълнението им базата данни е напълно готова за използване.
